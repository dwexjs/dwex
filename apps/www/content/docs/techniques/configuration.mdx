## Overview

Dwex makes it easy to work with environment variables and configuration using Bun's native APIs.

## Using Bun.env

```typescript
@Injectable()
export class ConfigService {
	get apiUrl() {
		return Bun.env.API_URL || "http://localhost:9929";
	}

	get port() {
		return Number(Bun.env.PORT) || 9929;
	}
}
```
