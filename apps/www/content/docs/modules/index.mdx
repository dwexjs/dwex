## Overview

Dwex provides official modules that add common functionality to your applications. These modules are optimized for Bun and follow the same patterns as your application code.

## Available Modules

### JWT Module (`@dwexjs/jwt`)

Implement JSON Web Token authentication with signing, verification, and token management.

```typescript
import { JwtModule } from "@dwexjs/jwt";

@Module({
	imports: [
		JwtModule.register({
			secret: process.env.JWT_SECRET,
			signOptions: { expiresIn: "1h" },
		}),
	],
})
export class AppModule {}
```

**Features:**
- Sign and verify JWT tokens
- Asymmetric key support (RS256)
- Custom claims and options
- TypeScript types included

### Logger Module (`@dwexjs/logger`)

Built-in logging with Pino for structured, performant logging.

```typescript
import { LoggerModule, Logger } from "@dwexjs/logger";

@Module({
	imports: [LoggerModule],
})
export class AppModule {}

@Injectable()
export class MyService {
	private readonly logger = new Logger(MyService.name);

	doWork() {
		this.logger.log("Work started");
		this.logger.error("Something failed", error);
	}
}
```

**Features:**
- Multiple log levels (debug, log, warn, error, fatal)
- Structured logging
- Pretty print for development
- File output support

## Installation

```bash
# JWT module
bun add @dwexjs/jwt

# Logger module
bun add @dwexjs/logger
```
